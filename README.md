## GETTING STARTED WITH INSTALLATION GO LANG 
* Step 1: Download the MSI installer from https://golang.org/dl/. 
* Step 2: Open `msi` file and follow the instruction to install it. 
* Step 3: Add `Go Lang` to path
* Step 4: Done!!! :zap:
